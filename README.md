# HyperCasualTemplate
Core Template for building a hypercasual game.

**Content:**

 - Full UI (Game, Win, Lose, Settings, Fonts, Animations)
 - Game States
 - Level Manager (use Resources)
 - Event Action Architecture
 - Sound and Haptic
 - Camera Management (Cinemachine)
 - DoTween
 - Pixelplacement
